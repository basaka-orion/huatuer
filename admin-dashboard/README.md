# 华图儿管理后台

华图儿AI创意绘画应用的管理后台，基于Next.js 14构建。

## 功能特性

- 📊 实时数据统计
- 👥 用户管理
- 🎨 作品管理
- 📈 数据分析
- ⚙️ 系统设置

## 技术栈

- **框架**: Next.js 14
- **样式**: Tailwind CSS
- **图表**: Recharts
- **图标**: Lucide React
- **语言**: TypeScript

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```
